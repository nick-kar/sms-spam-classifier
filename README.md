# SMS Spam Classifier

This project is a neural network model that classifies SMS messages as **ham** or **spam**.  
Built using TensorFlow and Python, the model uses an Embedding + LSTM architecture for sequence classification.

## Dataset
The dataset comes from [FreeCodeCamp SMS Spam Collection](https://cdn.freecodecamp.org/project-data/sms/) and contains training and validation data.
